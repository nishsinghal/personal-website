About Me

Hello! My name is Nishita and I'm a Digital Designer and aspiring Front-end Developer based in London. While I currently work primarily as a designer creating pixel-perfect designs in Photoshop, I love getting my hands dirty with all aspects of creating great web products, from thinking about the UX and fundamental organisation of the site to coding it up and bringing it to life.

I'm currently expanding my coding knowledge, learning the interactive delights of Javascript and j-query. I'm planning to follow this up with some back-end code as I think it's really important to fully appreciate how a website is put together.

I'm excited about really pushing the boundaries of the web to create beautiful, easy to use interfaces. I'm especially excited about responsive design and the challenge of creating something that is both usable as well as attractive across a range of devices.
